LeetCode 53. Maximum Subarray

Posted on 2022-11-16

LeetCode 53. Maximum Subarray

Main Idea

暴力就是枚举长度为1..sum.size()的所有子数组，然后求和，取最大值。

1 + 2 + … + k = O(n ^ 2)

这里很容易能够感觉到有状态转移的感觉，有些长度为k的子数组的和，可以由长度为k-1的子数组的和转移过来

dp[i] 表示所有以i结尾的子数组的和的最大值

划分区间的方式是长度

区间长度>=2和区间长度=1(nums[i] 自己)

区间长度>=2的区间即为f[i - 1] + nums[i]

区间长度=1的区间即为nums[i]

dp[i] = max(dp[i - 1] + nums[i], nums[i]) = nums[i] + max(dp[i - 1], 0)

因为这里只需要上一次的状态，所以可以用一个变量来代替dp数组

Follow up: If you have figured out the O(n) solution, try coding another solution using the divide and conquer approach, which is more subtle.

LeetCode 47. Permutations II

Main Idea

暴搜是应该的，搜索顺序以位置为主，而且以位置为搜索顺序能保证字典序

这题需要保证unique

所以我们，如果为每个数编号，那么只要保证枚举的时候各个数相对位置不变
比如三个1分别标记上1 2 3,我们规定只能从编号1枚举到编号3,不能132
保证这样的话，就只会有一个112

举个例子，有数112,可能会产生1(1)1(2)2 和 1(2)1(1)2, 但是我规定了必须吸纳枚举编号1，所以只会有1(1)1(2)2

为了能更好地枚举递增编号的相同value的数，我们需要先排序, 让相同的数相邻

这样操作是至少O(nlogn)的。
