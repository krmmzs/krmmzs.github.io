Setup

install cuda

The CUDA Version is the highest version of cuda you support

Install python3.10.x

more see pyenv install

Install Stable Diffusion web UI

git clone from Stable Diffusion web UI

to autoinstall it.

It will automatically build a venv virtual environment and install
the package in it, because the Pytorch to be installed is still
relatively large, so it will take a long time.

If everything ok, will see

meaning need models.
