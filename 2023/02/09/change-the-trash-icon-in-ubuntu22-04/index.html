Where are all the icons in ubuntu22.04?

In Ubuntu, icons are stored in several directories, including:

  • /usr/share/icons: This directory contains system-wide icons that are installed by default as part of various packages.
  • /usr/local/share/icons: This directory contains icons that have been installed manually or by third-party software.
  • ~/.icons: This directory, located in the home directory of each user, contains custom icons that have been installed or created by individual users.

In addition to these directories, some applications may include their own icons in their installation directories. For example, icons for a specific application might be located in /usr/share/<application-name>/icons.

Note that the directories listed above may contain subdirectories, each of which may contain icons for a specific theme. For example, the /usr/share/icons directory may contain subdirectories such as Adwaita, Humanity, elementary, etc. each containing icons for a specific theme.
